Welcome to our updated documentation! This guide contains everything you need to know to get started with your issued kit.

<Warning>
  Handling batteries incorrectly can be dangerous. Make sure you follow [these](/Batteries) safety guidelines at all times.
</Warning>

## Kit Checklist

Please check that you have received all of the following items:

### Electronic Kit

- 1x RoboCon Brain
- 1x Power switch (latching, red switch, black connector)
- 1x Start Button (momentary, black switch, green connector)
- 1x 6-pin GPIO connector
- 1x 2-pin 12V Accessory connector
- 2x 2-pin motor connector

### Batteries

- 2x 3S (11.1V nominal) LiPo Battery
- 1x B3 PRO imaxRC Compact LiPo Charger
- 1x “LiPo Safe” bag for storage and charging

### Computer

- 1x Linux Laptop
- 1x Power brick and lead

### Components

- 1x Components box
- 1x IR sensor
- 1x RGB LED
- 1x LED
- 1x Contact switch
- 2x Male to male connectors
- 4x Male to female connectors
- 1x Ultrasonic sensor
- 1x Pink micro servo (9g MG90S)

### Hardware & Other

- 1x 12L Really Useful Box
- 1x Really Useful Box Lid
- 1x Screwdriver set
- 1x Robot perspex kit
- 1x Cardboard box with AprilTag
- 1x Retrospective tape sample
- 2x Motors with connectors
- 12x AprilTags
